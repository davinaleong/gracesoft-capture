# Shared Inbox Performance Upgrades

This release focused on speed in the areas your team touches most.

## Improvements included

- Faster first render for inbox lists with cached status facets.
- Reduced query overhead when opening a message with notes and replies.
- More predictable pagination behavior for very large inboxes.
